# HOSTS-DATEI ANLEITUNG
# ===============================================
#
# Diese Datei muss zur Windows hosts-Datei hinzugefügt werden:
# C:\Windows\System32\drivers\etc\hosts
#
# WICHTIG: Als Administrator öffnen!
#
# ===============================================

# Klubportal Multi-Tenancy Subdomains
127.0.0.1       testclub.localhost
127.0.0.1       localhost

# Weitere Clubs können hier hinzugefügt werden:
# 127.0.0.1       anotherclub.localhost
# 127.0.0.1       myclub.localhost

# ===============================================
# ANLEITUNG ZUM BEARBEITEN DER HOSTS-DATEI:
# ===============================================
#
# OPTION 1 - Editor als Administrator:
# 1. Windows-Taste drücken
# 2. "Notepad" eingeben
# 3. RECHTSKLICK auf "Notepad" → "Als Administrator ausführen"
# 4. Datei → Öffnen → C:\Windows\System32\drivers\etc\hosts
# 5. Im Dateiauswahlfenster "Alle Dateien (*.*)" wählen
# 6. Datei hosts öffnen
# 7. Zeile "127.0.0.1       testclub.localhost" am Ende hinzufügen
# 8. Speichern (Strg+S)
# 9. Notepad schließen
#
# OPTION 2 - PowerShell als Administrator:
# 1. Windows-Taste + X drücken
# 2. "Windows PowerShell (Administrator)" wählen
# 3. Folgenden Befehl ausführen:
#
#    Add-Content -Path C:\Windows\System32\drivers\etc\hosts -Value "`n127.0.0.1       testclub.localhost"
#
# 4. PowerShell schließen
#
# ===============================================
# PRÜFEN OB ES FUNKTIONIERT:
# ===============================================
#
# 1. PowerShell öffnen
# 2. Befehl ausführen: ping testclub.localhost
# 3. Sollte antworten mit "Antwort von 127.0.0.1"
#
# ===============================================
# NACH BEARBEITUNG:
# ===============================================
#
# 1. Cache leeren: php artisan optimize:clear
# 2. Server neu starten: php artisan serve
# 3. Browser öffnen: http://testclub.localhost:8000/club
#
